У девочки есть безразмерные листы пленки и неокрашенной (бесцветной)
бумаги. Из них она может вырезать различные геометрические фигуры (круг,
прямоугольник, прямоугольный треугольник.) определенного размера, причем из
одних фигур можно вырезать другие. Вырезаемая фигура должна быть меньше той,
из которой ее вырезают. Еще у девочки есть набор красок для фигур (красная,
зеленая, синяя). Бумажные фигуры можно красить, но только 1 раз. Фигуры из
пленки бесцветные и красить их нельзя. Так же, у девочки есть коробка на 20
фигур, куда можно складывать и доставать любые фигуры.
Создать консольное приложение:
1. Использовать возможности ООП: классы, наследование, полиморфизм,
инкапсуляция.
2. Использовать возможности интерфейсов Java
3. Использовать возможности Enum.
4. Каждый класс должен иметь исчерпывающие смысл, название и
информативный состав.
5. В классах должны быть методы toString(), hashCode() и equals().
6. Классы должны быть грамотно разложены по пакетам.
7. Исходные данные считывать из текстового файла
